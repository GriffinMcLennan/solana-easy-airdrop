# Web

Example React application demonstrating `@solana-easy-airdrop/react` SDK integration.

## Setup

```bash
pnpm install
cp .env.example .env
```

## Development

```bash
pnpm dev
```

Runs on http://localhost:5173.

## Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `VITE_AIRDROP_SERVER_URL` | URL of the airdrop proofs server | http://localhost:5000 |
